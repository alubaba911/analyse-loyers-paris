# Analyse des Loyers à Paris

Projet d'analyse de données portant sur les loyers de référence à Paris à partir d'un dataset public sur l'encadrement des loyers.

## Objectif

Analyser les facteurs qui influencent les loyers au m² à Paris et construire un modèle capable de prédire ces loyers.

## Étapes de l'analyse

1. Chargement et exploration du dataset avec **Pandas**
2. Nettoyage des données et suppression des variables inutiles
3. Analyse exploratoire des données (EDA)
4. Visualisations avec **Matplotlib et Seaborn**
5. Encodage des variables catégorielles
6. Construction d'un **modèle de régression linéaire**
7. Évaluation du modèle avec le **score R²**

## Analyses réalisées

- Distribution des loyers de référence
- Impact du **nombre de pièces** sur le loyer
- Différences selon le **type de location**
- Analyse par **quartier**
- Étude des corrélations entre variables

## Modélisation

Un modèle de **régression linéaire (Scikit-learn)** est utilisé pour prédire les loyers de référence à partir des caractéristiques du logement.

## Technologies utilisées

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset

Dataset public sur l'encadrement des loyers à Paris.
